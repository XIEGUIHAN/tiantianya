隐私政策

最后更新日期：2026年2月7日

一、引言

欢迎使用「填填鸭网申神器-智能简历自动填写助手」（以下简称"本扩展"）。我们非常重视您的隐私保护。本隐私政策旨在向您说明我们如何收集、使用、存储和保护您的个人信息。

使用本扩展即表示您同意本隐私政策中描述的数据处理方式。如果您不同意本政策，请停止使用本扩展。

二、我们收集的信息

为了提供自动填写服务，我们会收集以下信息：

  • 注册信息：您的电子邮箱地址，用于账户注册、登录和验证码发送。
  • 简历数据：您主动输入或导入的简历信息（如姓名、教育经历、工作经历等），用于自动填写网申表单。
  • 用户反馈：您通过反馈功能提交的页面网址和问题描述，用于改进产品。
我们不会收集您的浏览历史、密码、银行卡信息或任何与本扩展功能无关的个人数据。

三、信息的使用方式

我们收集的信息仅用于以下目的：

  • 提供核心服务：将您的简历数据与网申页面表单字段进行智能匹配和自动填写。
  • 产品改进：分析用户反馈，优化匹配算法和适配更多招聘平台。
  • 客户支持：响应您的问题和反馈。

四、信息的存储与安全

  • 存储位置：您的数据存储在位于中国大陆的安全服务器上。
  • 传输安全：所有数据通过 HTTPS 加密传输，防止传输过程中被截获。
  • 访问控制：仅授权的开发人员可以访问服务器数据，且仅限于维护和故障排查目的。
  • 本地存储：部分数据（如登录状态）通过浏览器的 Chrome Storage API 存储在您的本地设备上。

五、信息的共享与披露

我们不会出售、出租或以任何商业目的向第三方分享您的个人信息。

仅在以下情况下，我们可能会披露您的信息：

  • 法律法规要求或政府机关依法要求时。
  • 为保护本扩展、用户或公众的权利、财产或安全所必需时。

六、您的权利

您对自己的个人信息享有以下权利：

  • 查看和编辑：您可以随时在扩展内查看和修改您的简历数据。
  • 删除数据：您可以联系我们删除您的账户及所有相关数据。
  • 撤回同意：您可以随时停止使用本扩展并卸载它。

七、Cookie 和追踪技术

本扩展不使用 Cookie 或任何第三方追踪技术。我们不会追踪您的浏览行为。

八、未成年人保护

本扩展主要面向求职者，不针对 16 岁以下的未成年人。如果我们发现无意中收集了未成年人的信息，将立即删除相关数据。

九、隐私政策的变更

我们可能会不时更新本隐私政策。更新后的政策将在本页面发布，并更新"最后更新日期"。建议您定期查看本页面以了解最新的隐私保护措施。
